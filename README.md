# Recipe-book

Приложение для создания, хранения и управления рецептами

## Состав

Инфтерфейс - PyQt6
Работа с базой данных - Sqlite3

## Особенности

- Drag and Drop для элементов некоторых списков
- Возможность редактирования содержимого элементов этих списков
- Несколько тем + тема, которую вы можете написать сами
